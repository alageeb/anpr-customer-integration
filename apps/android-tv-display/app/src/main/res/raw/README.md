This directory should contain sound files for the ANPR Display app.

Required files:
- welcome.mp3 : Sound played when a customer checks in (welcome screen)
- call.mp3    : Sound played when a ticket is called

To generate placeholder sounds, you can:
1. Use any short notification sound (1-2 seconds)
2. Convert to MP3 format
3. Place in this directory

For production:
- Use appropriate notification sounds for TV environment
- Keep sounds short (1-2 seconds max)
- Ensure volume is appropriate for public display
